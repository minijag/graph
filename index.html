Part 2

  1. Write a reusable component that manages the selection state of products currently shown in the chart. Internally, the products are represented by unique string ids (Symbol). The following operations need to be supported:
    • Changing the selection status for a product
    • Querying the selection status for a product
    • Setting the list of displayed products
    • The component should provide a means for clients to receive notifications whenever the selection state of a product changes.
